# QuanLyHocTapAI
Trợ Lý Học Tập AI: Tối Ưu Hóa Lịch Trình &amp; Hiệu Suất.  Dự án Python/Flask xây dựng hệ thống AI Phản Hồi (Nudging). Sử dụng Logic Ưu tiên để tự động xếp lịch và sắp xếp nhiệm vụ. Tích hợp Chatbot thông minh và Cơ chế Đẩy (Push Mechanism) để cảnh báo tiến độ, định hướng hành vi học tập. Mục tiêu: Tối đa hóa hiệu suất bằng công nghệ Rule-Based AI.
